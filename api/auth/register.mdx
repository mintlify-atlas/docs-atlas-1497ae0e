title: Register user

<Warning>
**Configuration Required**: This endpoint requires [AshJsonApi setup](/api/setup) before it will work. By default, Ash Coffee uses browser-based authentication.
</Warning>

## Endpoint

```
POST /api/json/users
```

Register a new user account with email and password. Upon successful registration, a JWT token is generated and returned in the response metadata.

<Note>
After registration, users must confirm their email address before they can access protected resources. See [Email Confirmation](/authentication/email-confirmation) for details.
</Note>

## Request

### Headers

<ParamField header="Content-Type" type="string" required>
  Must be `application/vnd.api+json`
</ParamField>

### Body Parameters

<ParamField body="data" type="object" required>
  The request data object
  
  <Expandable title="properties">
    <ParamField body="type" type="string" required>
      Must be `"user"`
    </ParamField>
    
    <ParamField body="attributes" type="object" required>
      User attributes
      
      <Expandable title="properties">
        <ParamField body="email" type="string" required>
          User's email address (case-insensitive)
        </ParamField>
        
        <ParamField body="password" type="string" required>
          User's password in plain text. Must be at least 8 characters long.
        </ParamField>
        
        <ParamField body="password_confirmation" type="string" required>
          Password confirmation. Must match the password field.
        </ParamField>
      </Expandable>
    </ParamField>
  </Expandable>
</ParamField>

## Response

### Success Response (201 Created)

<ResponseField name="data" type="object">
  The created user resource
  
  <Expandable title="properties">
    <ResponseField name="id" type="string">
      Unique user identifier (UUID)
    </ResponseField>
    
    <ResponseField name="type" type="string">
      Resource type: `"user"`
    </ResponseField>
    
    <ResponseField name="attributes" type="object">
      User attributes
      
      <Expandable title="properties">
        <ResponseField name="email" type="string">
          User's email address
        </ResponseField>
        
        <ResponseField name="confirmed_at" type="string | null">
          Email confirmation timestamp (ISO 8601). `null` if not yet confirmed.
        </ResponseField>
      </Expandable>
    </ResponseField>
    
    <ResponseField name="meta" type="object">
      Response metadata
      
      <Expandable title="properties">
        <ResponseField name="token" type="string">
          JWT authentication token that can be used for subsequent requests
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

## Examples

<CodeGroup>
```bash cURL
curl -X POST https://api.example.com/api/json/users \
  -H "Content-Type: application/vnd.api+json" \
  -d '{
    "data": {
      "type": "user",
      "attributes": {
        "email": "user@example.com",
        "password": "secure_password_123",
        "password_confirmation": "secure_password_123"
      }
    }
  }'
```

```javascript JavaScript
const response = await fetch('https://api.example.com/api/json/users', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/vnd.api+json',
  },
  body: JSON.stringify({
    data: {
      type: 'user',
      attributes: {
        email: 'user@example.com',
        password: 'secure_password_123',
        password_confirmation: 'secure_password_123'
      }
    }
  })
});

const result = await response.json();
const token = result.data.meta.token;

// Store token for future requests
localStorage.setItem('authToken', token);
```

```python Python
import requests

response = requests.post(
    'https://api.example.com/api/json/users',
    headers={'Content-Type': 'application/vnd.api+json'},
    json={
        'data': {
            'type': 'user',
            'attributes': {
                'email': 'user@example.com',
                'password': 'secure_password_123',
                'password_confirmation': 'secure_password_123'
            }
        }
    }
)

result = response.json()
token = result['data']['meta']['token']

# Store token for future requests
print(f"Token: {token}")
```
</CodeGroup>
