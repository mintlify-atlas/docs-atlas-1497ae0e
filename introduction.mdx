  <h2 className="text-2xl sm:text-3xl font-bold text-gray-900 dark:text-white mb-4">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get your authentication API up and running in four simple steps.
  </p>

  <Steps>
    <Step title="Clone and set up the project">
      Clone the repository and install dependencies:

      ```bash
      git clone https://github.com/MmmarRTha/backend_ash_coffee.git
      cd backend_ash_coffee
      mix setup
      ```

      This installs all dependencies, creates the database, runs migrations, and seeds initial data.
    </Step>

    <Step title="Configure environment variables">
      Create a `.env` file or set the required environment variables:

      ```bash
      export DATABASE_URL="postgresql://user:password@localhost/ash_coffee_dev"
      export SECRET_KEY_BASE="your-secret-key-base"
      ```

      Generate a secret key base with:

      ```bash
      mix phx.gen.secret
      ```
    </Step>

    <Step title="Start the Phoenix server">
      Launch the development server:

      ```bash
      mix phx.server
      ```

      The API will be available at `http://localhost:4000`. The OpenAPI documentation is accessible at `http://localhost:4000/api/json/swaggerui`.
    </Step>

    <Step title="Register your first user">
      Make your first API request to register a new user:

      ```bash
      curl -X POST http://localhost:4000/api/json/auth/register \
        -H "Content-Type: application/vnd.api+json" \
        -d '{
          "data": {
            "type": "user",
            "attributes": {
              "email": "test@example.com",
              "password": "secure_password_123",
              "password_confirmation": "secure_password_123"
            }
          }
        }'
      ```

      <Accordion title="Example Response">
        ```json
        {
          "data": {
            "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
            "type": "user",
            "attributes": {
              "email": "test@example.com",
              "confirmed_at": null
            },
            "meta": {
              "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
            }
          }
        }
        ```
      </Accordion>
    </Step>
  </Steps>
